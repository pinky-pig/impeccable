常见问题

下载后的文件应该放在哪里？

最简单的方法是执行 npx skills add pbakaus/impeccable，它会自动识别你的 AI 工具并把文件放到正确位置。

如果你下载的是 通用 ZIP，请将它解压到你的项目根目录（和 package.json 或 src/ 同级）。它会生成各工具所需的隐藏目录：.cursor/、.claude/、.gemini/、.codex/ 和 .agents/。

项目级安装优先级更高，也更方便你把 skills 纳入版本控制。

命令或技能没有显示，怎么办？

对于命令：在你的 AI 工具里输入 /，查看是否出现 /audit、/polish 等命令。如果没有，请先确认文件位置是否正确。

对于技能：技能会在相关场景下自动触发。想验证是否生效，可以在提示词中明确写上 “use the frontend-design skill”，这样模型会显式识别并应用它。

工具特定设置：

  • Cursor：需要切到 Nightly 频道，并在 Settings → Rules 中启用 Agent Skills
  • Gemini CLI：需要安装 @google/gemini-cli@preview，并通过 /settings 启用 Skills
我是第一次接触这些 AI 工具，应该从哪里开始？

Skills 和 commands 更偏进阶能力。如果你刚开始接触，建议先掌握基础用法：

  • Claude Code：官方文档
  • Cursor：Cursor 文档
  • Gemini CLI：Gemini CLI 文档
  • Codex CLI：Codex GitHub

等你熟悉了基础提示词和代码生成流程，再回来试试 Impeccable，会更容易发挥它的价值。
