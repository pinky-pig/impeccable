隐私说明

最后更新：2026 年 4 月 2 日

Impeccable 是什么

Impeccable 是一组开源的 agent skills（文本文件），运行在你的本地 AI 编程工具里。skills 本身不会收集数据、不会主动发起网络请求，也不内置分析追踪。

网站分析

本站（www.impeccable.cn）使用 Google Analytics 和 Umami 来了解访问情况，例如页面浏览量、来源和大致地区。除分析工具默认提供的数据外，我们不会额外收集个人信息，也不会将这些数据用于广告投放。

下载记录

当你从网站下载 skill bundle 时，站点可能记录下载事件（例如下载了哪个 bundle、时间戳），用于统计使用情况。这些记录不附带你的个人身份信息。

Claude Code 插件

当以 Claude Code 插件形式安装时，Impeccable 完全运行在你的本地 Claude Code 会话中。数据不会发送到 Impeccable 的服务器。Claude Code 应用本身受 Anthropic 自己的隐私政策约束。

GitHub

项目源码托管在 GitHub 上。你与仓库的交互（issues、pull requests、stars 等）受 GitHub 隐私政策 约束。

联系

如果你对这份说明有疑问，可以在 GitHub 上提交 issue。与原项目本身相关的隐私问题，也可以参考原作者公开的信息渠道。
